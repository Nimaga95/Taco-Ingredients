# Taco-Ingredients

A brief project to review basics notions of JSON, EJS et Node js. 

Dowload the project then in your terminal tape :

1. npm i -y
2. npm i express
3. npm i body-parser
4. npm i json
5. node index.js to go to the web site (https://localhost:3000/.
